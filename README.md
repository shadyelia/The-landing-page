# The-landing-page

- Steps to run: <br />
    1- install npm cli using: `npm install -g npm`<br />
    2- Install live-server using: `npm install live-server -g`<br />
    3- Start the server using: `live-server` and it runs on `http://127.0.0.1:8080/`<br />

<br />
<br />
